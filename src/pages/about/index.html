What is Islands Architecture?

Islands architecture is a modern web development pattern that combines the best of static site generation with dynamic, interactive components.

Key Concepts:

  • Server-Rendered HTML: Most of the page is static or server-rendered
  • Selective Hydration: Only interactive components ("islands") load JavaScript
  • Independent Islands: Each island hydrates independently and asynchronously
  • Performance First: Dramatically reduced JavaScript bundle sizes
